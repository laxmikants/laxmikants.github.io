Tex formulas

Using TeX

 Posted on March 5, 2017  |  2 minutes  |  234 words  |  Laxmikant Soni

TeX can be used to code complex statistical formulas.

Mean

\(\bar{x} = {\frac{1}{n} \sum\limits_{i=1}^{n} x_i }\)

$$(\bar{x} = {\frac{1}{n} \sum\limits_{i=1}^{n} x_i })$$

population variance

\(\sigma^2 = {\frac{1}{n} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }\)

$$(\sigma^2 = {\frac{1}{n} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 })$$

sample variance

\(s^2 = {\frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }\)

$$(s^2 = {\frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 })$$

Population Standard Deviation

\(\sigma = \sqrt{\sigma^2} = \sqrt{{\frac{1}{n} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }}\)

$$(\sigma = \sqrt{\sigma^2} = \sqrt{{\frac{1}{n} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }})$$

Sample Standard Deviation

\(s = \sqrt{s^2} = \sqrt{{\frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }}\)

$$(s = \sqrt{s^2} = \sqrt{{\frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})^2 }})$$

Covariance

\(Cov(x,y) = \frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y}) \)

$$(Cov(x,y) = \frac{1}{n - 1} \sum\limits_{i=1}^{n} (x_i - \bar{x})(y_i - \bar{y}) )$$

Z-score population

\(z = \frac{x_i - \mu}{\sigma} \)

$$(z = \frac{x_i - \mu}{\sigma} )$$

Z-score Sample

\(z = \frac{x_i - \bar{x}}{s} \)

$$(z = \frac{x_i - \bar{x}}{s} )$$

Chi-square

\chi^2 = \sum \frac {(O - E)^2}{E}

$$\chi^2 = \sum \frac {(O - E)^2}{E}$$

Linear Regression

\hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_i + \hat{\epsilon}_i

$$\hat{Y}_i = \hat{\beta}_0 + \hat{\beta}_1 X_i + \hat{\epsilon}_i$$

Slope

\hat{\beta}_1 = \frac{\sum(X_i – \bar{X}) (Y_i – \bar{Y})} {\sum(X_i – \bar{X})^2}

$$\hat{\beta}_1 = \frac{\sum(X_i – \bar{X}) (Y_i – \bar{Y})} {\sum(X_i – \bar{X})^2}$$

Intercept

\hat{\beta}_0 = \bar{Y} – \hat{\beta}_1 \bar{X}

$$\hat{\beta}_0 = \bar{Y} – \hat{\beta}_1 \bar{X}$$
